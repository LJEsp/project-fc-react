# BaseButton

## Props

| Prop        | Type    | Default | Options         | Description          |
| ----------- | ------- | ------- | --------------- | -------------------- |
| `width`     | String  | `null`  | `fill`, `icon`  | Set width style      |
| `size`      | String  | `null`  | `small`         | Set size             |
| `isLoading` | Boolean | `null`  | `true`, `false` | Set state to loading |
